# Four Actions Framework

The Four Actions Framework, also called the ERRC Grid, is the core value-innovation tool in Blue Ocean Strategy. It helps teams redraw the industry's value curve instead of fighting on the same crowded one.

## Why It Matters

Most companies compete on the same features, then wonder why margins shrink. ERRC asks a different question: what should be eliminated, reduced, raised, or created?

## The Four Actions

| Action | Question | What It Means |
|---|---|---|
| Eliminate | What does the industry take for granted that customers do not value? | Remove unnecessary cost and complexity |
| Reduce | What should be offered below industry standard? | Cut overdesigned or wasteful elements |
| Raise | What should be lifted above industry standard? | Improve the attributes customers truly care about |
| Create | What has never existed in the category? | Define a new source of value |

## When to Use It

Use ERRC when the market feels trapped in price wars, feature sameness, or declining margins. It is a strategy tool, not a day-to-day operating tool.

## Cirque du Soleil Case Study

Cirque du Soleil redefined the circus industry:

- Eliminated animal acts
- Reduced reliance on star performers
- Raised the artistic and theatrical quality
- Created a tent-based touring format with a new audience experience

The result was a new value curve that looked nothing like the traditional circus.

## How to Run an ERRC Analysis

1. Map the current industry value curve.
2. List the main factors customers and competitors care about.
3. Ask what can be eliminated.
4. Ask what can be reduced.
5. Ask what can be raised.
6. Ask what can be created.
7. Test the new curve with an MVP or pilot.

## Common Mistakes

| Mistake | Why It Fails |
|---|---|
| Treating eliminate as simple cost cutting | The goal is to redefine value, not just slash budgets |
| Using reduce instead of innovating | Reduction should create focus, not compromise |
| Treating raise as feature bloat | Raise should improve something customers actually value |
| Treating create as a small tweak | Create should introduce a genuinely new value dimension |
