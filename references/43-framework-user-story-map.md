# User Story Map

Jeff Patton's user story map organizes product work around the user's journey instead of a flat backlog. It helps teams see the whole experience and slice the MVP horizontally and vertically.

## Why a Flat Backlog Fails

A traditional backlog is just a list. It is easy to sort, but hard to understand in context.

User story mapping turns the list into a story: a complete end-to-end journey broken into activities, tasks, and stories.

## The Three Layers

| Layer | Meaning |
|---|---|
| Activities | The big user goals or stages in the journey |
| Tasks | The concrete actions inside each activity |
| Stories | The detailed stories that can be prioritized and delivered |

Horizontal movement shows the sequence of the journey. Vertical movement shows priority.

## When to Use It

User story maps are useful when:

- A journey has many steps or touchpoints
- You need to prioritize requirements from the user's point of view
- Different teams need a common visual language
- You are planning agile sprints
- You want to reshape an existing product
- You need a roadmap grounded in user value

## Salesforce Case Study

Salesforce used story mapping to redesign Sales Cloud and reconnect the sales workflow into one coherent journey.

The product team mapped the path from lead generation to qualification, proposal, closing, and reporting. The result was a more connected workflow, higher efficiency, and better customer satisfaction.

### Example flow

| Prospecting | Qualifying | Proposing | Closing | Reporting |
|---|---|---|---|---|
| Lead sourcing | Customer screening | Needs analysis | Contract generation | Sales reporting |
| Lead assignment | Customer scoring | Proposal recommendation | Approval flow | Performance reporting |

## How to Make the Map Useful

1. Start from the user journey, not the feature list.
2. Organize by user needs instead of team boundaries.
3. Use vertical slices to define the smallest valuable release.
4. Make sure each release delivers a testable outcome.
5. Use the map to plan iteration rhythm and release scope.
