# Task Completion Rate

Task Completion Rate is a usability metric that measures whether users can actually complete the tasks your product asks them to do. If users cannot finish the task, the product is not working well enough.

## Why It Matters

A product can look polished and still fail badly in practice.

If users do not know how to use the product, the problem is not their intelligence. The problem is the interface.

Nielsen Norman Group's research suggests that even simple websites can be hard to use the first time, and mobile interfaces can be worse because the screen is smaller and the flow is more constrained.

## What TCR Measures

| User behavior | Completion | Satisfaction | Ease of use | Overall rating |
|---|---|---|---|---|
| Task completed | Yes | Yes | Yes | Excellent |
| Partially completed | Partial | Partial | Partial | Good |
| Failed | No | No | No | Poor |

## When to Use It

Task Completion Rate works for almost any product with user flows:

- Pre-launch acceptance testing
- A/B comparison of two designs
- Problem diagnosis
- Competitive benchmarking
- Iteration validation
- ROI analysis

## Gowuk Case Study

The UK Government Digital Service used TCR-style testing to improve GOV.UK services and prove the impact of their work.

### Results

| Metric | Before | After |
|---|---|---|
| Task completion rate | 48% | 82% |
| Average completion time | 4.5 minutes | 2.1 minutes |
| User satisfaction | 60% | 85% |
| Annual operating cost | Higher | Reduced by about £35M |

### Improvements made

- Simplified the form
- Reduced required fields from 15 to 8
- Added progress indicators
- Improved error messages

## How to Run a TCR Test

1. Design the tasks.
   - Pick representative tasks that reflect the product's core flows.

2. Recruit users.
   - Choose 5-8 real target users.

3. Run the test.
   - Let users work independently and observe where they struggle.

4. Analyze the data.
   - Calculate completion rate, time, and satisfaction.

5. Write the report.
   - Summarize the problems and recommend fixes.
