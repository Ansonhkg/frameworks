# Growth Flywheel

The growth flywheel turns core user behavior into a self-reinforcing loop. More users create more value, which attracts more users, which creates compounding growth.

## Why Linear Growth Breaks

Ad-driven growth works until the ads stop.

The flywheel model asks a better question: what mechanism makes the product better as more people use it?

## Two Classic Flywheels

### Amazon

More users -> more sellers -> more selection -> better experience -> more traffic -> lower prices -> more users

### SaaS example

More users -> more data -> better recommendations -> better experience -> higher retention -> more users

## When to Use It

Flywheels are useful both as a design tool and as a diagnostic tool.

| Product type | Why the flywheel matters |
|---|---|
| User-driven products | User actions directly create value |
| Network-effect products | More users make the product more valuable |
| Data-driven products | Usage improves the product over time |
| Content/community products | User contributions attract new users |

## Jeff Bezos and the Napkin

After the dot-com crash, Jeff Bezos sketched Amazon's flywheel on a napkin to show how the business engine worked.

The insight was simple: the platform becomes stronger as the loop continues to turn.

## Six Common Mistakes

1. Mistaking a linear process for a flywheel.
2. Ignoring network effects.
3. Picking weak or unmeasurable nodes.
4. Forgetting the startup cost of getting the wheel moving.
5. Oversimplifying a real business system.
6. Ignoring outside forces such as competition and technology shifts.
