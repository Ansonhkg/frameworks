# North Star Metric

The North Star Metric is the single metric a product team uses to measure the core value it creates. It aligns the whole team around one measurable outcome.

## Introduction

When every team is optimizing something different, value disappears into local metrics. The North Star Metric gives everyone a common direction.

## Three-Layer Metric System

The North Star is not the only metric. It sits at the top of a stack:

- North Star Metric: the one metric that best reflects core value
- Input Metrics: the user behaviors that drive the North Star
- Output Metrics: the direct results of user behavior

## What to Avoid

| Good metrics | Poor metrics |
|---|---|
| North Star Metric | Vanity metrics |
| Input metrics | Lagging metrics |
| Output metrics | Metrics that look good but do not guide action |

## Examples

| Company | North Star Metric |
|---|---|
| Airbnb | Nights booked |
| Spotify | Daily active users |
| Facebook | Monthly active users |
| Uber | Completed rides |
| LinkedIn | Professional interactions |
| Twitter | Daily tweets |

## When to Use It

North Star thinking is especially useful early in product development, when the team needs focus. As the product matures, the metric system can become more detailed.

## How to Set One

1. Identify the core value your product creates.
2. Define the key behavior that reflects that value.
3. Make the metric measurable.
4. Keep improving the product based on the data.

## Spotify Case

Spotify's North Star metric has evolved as the business matured.

| Year | Metric |
|---|---|
| 2015 | Daily active users |
| 2017 | Monthly active users |
| 2019 | Monthly listening time |
| 2021 | Retention |

The shift reflects a change in strategic focus from usage growth to deeper engagement.

## Five Reminders

1. Do not over-optimize a single metric.
2. Revisit the North Star regularly.
3. Make the metric actionable.
4. Align it with business goals.
5. Keep it simple enough for the team to remember.
